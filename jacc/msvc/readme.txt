Jacc.dsw is a
Microsoft Visual Studio 6 Workspace file
It contains the following projects:
  - Jacc.dsp   - project to build the Jacc tool itself;
  - Calc.dsp   - example1 - simple command line calculator;
  - Java12.dsp - example2 - complete java 1.2 parser (w/o semantic analysis!);

Jacc.sln is a
Microsoft Visual Studio 7 Solution file
It contains the following projects:
  - Jacc.vcproj   - project to build the Jacc tool itself;
  - Calc.vcproj   - example1 - simple command line calculator;
  - Java12.vcproj - example2 - complete java 1.2 parser (w/o semantic analysis!);
