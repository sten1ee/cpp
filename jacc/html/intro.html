A Brief Historical Note

I started Jacc as a hobby project. It would be more precise to say: there was this hobby project (started in the middle of 1998) that led to Jacc. Finally my MSc thesis (18 November 1999 at Sofia University, faculty of Mathematics and Informatics ) was build around Jacc.

Jacc's user manual (which is part of my thesis) is written in Bulgarian and to my regret it is not 'ported' to English yet. Fortunately the LALR parser generators are popular enough so if you have some experience with tools like Yacc, Bison or Java CUP it won't be difficult to understand most of Jacc. A couple of examples is added (java12 parser and command line calculator - calc) to further clarify the issue.

The name 'J-A-C-C'

As one can see (using a simple AltaVista search) the name 'jacc' is quite heavily exploited. And some of these products are compiler-compilers too. Just to name a few:
  • Invisible Jacc Home Page
  • Hassan Ait-Kaci's Jacc
The product introduced on this page is developed independantly from the two above mentioned and any other Jacc-named product. It just shares the same name. In order to resolve naming conflicts the author would like to introduce the product namespace 'jerk0main' so that Jacc's full name should read jerk0main Jacc.
